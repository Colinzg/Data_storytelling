# 这一夜，我们家都没有睡好

> 一个中国三代同堂家庭的睡眠数据叙事作品

---

## 📖 项目简介

本项目通过一个普通中国城市家庭（外婆、父亲、母亲、儿子、姐姐）在同一夜晚的故事，结合真实睡眠调查数据，以网页长滚动叙事的方式呈现当代中国社会各年龄段的睡眠困境。

**核心洞察：** 原来睡不好，发生在每一代人身上，只是原因不同。

## 🎨 设计风格

- **配色：** 深夜蓝 + 暖黄（夜晚静谧感 + 灯光温暖感）
- **叙事方式：** 时间线推进，五幕结构
- **视觉参考：** 类似 SCMP 数据新闻的沉浸式滚动叙事

## 🏗️ 叙事结构

```
21:30  开场 —— 电视播报睡眠新闻，一家人不以为意
22:30  家庭介绍 —— 五口人各自准备入睡
23:30  第一幕 —— 儿子：睡眠被"未来"压缩（学业压力）
00:20  第二幕 —— 姐姐：报复性熬夜（手机依赖）
01:10  第三幕 —— 父母：睡眠被责任打断（工作+家务）
04:50  第四幕 —— 外婆：被年龄叫醒（老年睡眠障碍）
07:00  第五幕 —— 早餐桌：所有人都困，但没人谈睡眠
结尾  —— 同一个屋檐下，每个人都在失眠
```

## 📊 数据来源

| 来源 | 内容 |
|------|------|
| 国家卫健委《健康中国行动（2019-2030）》 | 各年龄段推荐睡眠时间 |
| 北京大学 CFPS 中国家庭追踪调查 | 中国人十年睡眠变化趋势 |
| 中国睡眠研究会 × 华为《2025中国睡眠健康研究白皮书》 | 超15万用户睡眠数据 |
| 中国社会科学院《2024年中国睡眠指数报告》 | 6586人样本调查 |
| 瑞思迈 ResMed《2026年全球睡眠调查》 | 全球睡眠障碍数据 |

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript，零依赖运行
- [Chart.js](https://www.chartjs.org/)（CDN 引入）—— 数据可视化
- SVG 手绘风格插画 —— 场景氛围
- Intersection Observer API —— 滚动动画

## 🚀 快速开始

```bash
# 直接在浏览器中打开
open index.html

# 或用本地服务器（推荐，避免跨域问题）
npx serve .
# 或
python -m http.server 8080
```
